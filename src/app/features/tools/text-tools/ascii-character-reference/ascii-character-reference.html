What is ASCII?

ASCII (American Standard Code for Information Interchange) is a character encoding standard used to represent text in computers and electronic devices. Developed in the 1960s, ASCII assigns a unique numeric code to each character, making it possible for different computer systems to exchange information.

ASCII Code Ranges

settings

Control Characters (0-31)

Non-printable characters used for text formatting and device control, including NULL, TAB, LINE FEED, and CARRIAGE RETURN.

text_fields

Printable Characters (32-126)

Letters (A-Z, a-z), numbers (0-9), punctuation marks, and common symbols that can be displayed on screen or printed.

delete

DEL Character (127)

The delete control character, originally used to mark characters for deletion on paper tape systems.

extension

Extended ASCII (128-255)

Additional characters including accented letters, currency symbols, and graphical characters for various languages and applications.
