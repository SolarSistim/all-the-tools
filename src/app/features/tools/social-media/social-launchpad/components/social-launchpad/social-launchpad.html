info Social Media Launchpad Guide

Supported Platforms

  • X (Twitter): 280 character limit, 1-3 hashtags recommended
  • Facebook: 63,206 character limit, hashtags less effective
  • LinkedIn: 3,000 character limit, 3-5 hashtags recommended
  • Threads: 500 character limit, 1-3 hashtags recommended
  • Tumblr: No character limit, casual tone encouraged
  • Mastodon: 500 character limit, 3-5 hashtags recommended
  • Bluesky: 300 character limit, hashtags not used
  • Truth Social: 500 character limit, 1-5 hashtags recommended

How to Use

  • Select platforms - Click platform buttons on the right to select which social networks you want to post to
  • Add your URL - Paste the link you want to share and click "Fetch OG Data" to automatically pull title, description, and preview image (Limited to {{ ogFetchLimitPerMin }} fetches per minute)
  • Write your description - Craft your post content in the description field. It auto-saves as you type
  • Add hashtags - Type hashtags and press Enter, Space, or Comma to add them. Remove with the X button
  • Insert emojis - Click "Add Emoji" to browse categorized emojis or search for specific ones
  • Monitor character counts - Watch the character counter at the top to ensure you're within limits for each platform
  • Preview posts - Click any selected platform to see how your post will appear
  • Copy content - Use the Copy button to copy your formatted post to clipboard

Best Practices

  • Start with the most restrictive platform - Write for the platform with the lowest character limit first, then expand for others if needed
  • Use hashtags strategically - Different platforms favor different hashtag counts. Follow the recommendations shown in warnings
  • Check character counts - The most restrictive platform is marked with a warning icon. Stay under that limit for maximum reach
  • Fetch Open Graph data - Rich previews increase engagement. Always fetch OG data when sharing links
  • Test with emojis - Emojis can increase engagement but count as characters. Use them thoughtfully
  • Keep URLs at the end - Most platforms auto-shorten URLs and show previews, so place them after your main content
